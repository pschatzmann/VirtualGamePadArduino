Detailed Description

Represents the state of a gamepad thumbstick, including position and angle helpers.

Provides both Cartesian (x, y) and polar (angle, magnitude) representations. Includes helpers for ROS and standard angle conventions.

  • x: horizontal axis, -1.0 (left) to 1.0 (right)
  • y: vertical axis, -1.0 (down) to 1.0 (up)
  • angle(): radians, 0 = x-axis (forward)
  • angleROS(): radians, ROS convention (0 = forward/x, 90 = left/y, -90 = right/-y)
  • angleDeg(): degrees, 0 = x-axis (forward)
  • angleDegROS(): degrees, ROS convention (0 = forward/x, 90 = left/y, -90 = right/-y)
  • magnitude(): stick displacement, 0.0 (center) to 1.0 (edge)
